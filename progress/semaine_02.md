# Rapport de progression — Semaine 2
**Projet** : Système Multi-Agents de Service Client  
**Entreprise** : Vala Orange — Agadir, Maroc  
**Date** : Semaine du 20/04 au 24/04

Je vous informe de l’avancement actuel de notre projet de système intelligent de service client basé sur une architecture multi-agents.

Actuellement, nous avons terminé l’implémentation des deux premiers agents principaux :

**Agent de classification** : chargé d’analyser et de catégoriser les messages des utilisateurs (information, support technique, réclamation, etc.).
**Agent de sécurité** : chargé de détecter les messages sensibles, les tentatives de prompt injection ou tout contenu non sécurisé afin de protéger le système.

Ces deux modules fonctionnent correctement et sont déjà intégrés dans le flux principal du système via l’orchestrateur.

Nous allons maintenant continuer avec les autres agents, notamment le module RAG et l’agent de génération de réponse.

Nous restons à votre disposition pour toute remarque ou orientation.

Cordialement.
